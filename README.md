# Nexo

A simple web app for tracking tasks and projects, inspired by Notion. Built for fun and learning purposes.

## Overview

- Backend: ![.NET / ASP.NET Core](https://img.shields.io/badge/-ASP.NET Core-512BD4?style=flat&logo=dotnet&logoColor=white) ![PostgreSQL](https://img.shields.io/badge/-PostgreSQL-336791?style=flat&logo=postgresql&logoColor=white) ![JWT](https://img.shields.io/badge/JWT-black?style=flat&logo=JSON%20web%20tokens)
- Frontend: TBD  
- Database: Simple relational DB with Users, Tasks, Projects

## Structure

- `/NexoApi` – Contains the ASP.NET Core backend  
- `/NexoApp` – Contains the frontend (React/Angular/Blazor, TBD)
